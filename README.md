# 伊朗战争成本追踪 (Iran War Cost Tracker CN)

中文版实时战争费用计算器，基于 [iran-cost-ticker.com](https://iran-cost-ticker.com) 创意。

## 功能

- 💰 实时军费计数器（人民币/美元）
- 📰 最新进展新闻
- 📊 数据来源说明

## 数据来源

- SIPRI (斯德哥尔摩国际和平研究所)
- CSIS (战略与国际问题研究中心)
- 开源国防数据分析

## 部署

使用 GitHub Pages 托管：https://techfitmaster.github.io/iran-war-tracker/

## 注意事项

⚠️ 本网站数据为估算值，仅供研究参考。
